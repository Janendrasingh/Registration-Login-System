# Registration-Login-System
A simple console-based Registration and Login System built using C++. This project demonstrates basic file handling, object-oriented programming, and user authentication logic.


## 📌 Features
- 📝 User Registration
- 🔑 User Login Authentication
- 📂 File Handling using fstream
- 🔒 Password validation (minimum 8 characters)
- ⏳ Console-based loading effects using Sleep()
- 💾 Stores user credentials in a local file


## 🛠️ Technologies Used
- **C++**
- **Standard Libraries**
  - `iostream`
  - `fstream`
  - `sstream`
  - `windows.h`


## 🚀 How It Works
1. Registration  
- User enters a Login ID
- User sets a Password (minimum 8 characters)
- Data is saved into a file (Login.txt)
2. Login  
- User enters credentials
- Program reads file and validates input
- If matched → ✅ Access granted
- If not → ❌ Error message.  


## ❗ Limitations
- Passwords are stored in plain text (not secure)
- No encryption or hashing
- No input sanitization
- Limited UI (console-based)
